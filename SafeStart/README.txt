Please make a backup of /Avorion/data/ before installing.

To install just copy the data folder into /Avorion/ and accept any overwrites.

What this mod does:

1. It removes pirates attacking in the start sector. Allowing players to safely start up.
2. It marks the starting sector as a PVP neutral zone.

Possible bugs:
This mod modifies the generation of the startingsector.
It will not work for an existing game.
